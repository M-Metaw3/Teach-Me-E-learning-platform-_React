			
				
# TeachMe (Courses Platform)
#link https://m-metaw3.github.io/e-l-react/
#### A website with an admin dashboard that enables the user to sign up and enroll in courses. 

## Technology Used
- HTML5
- CSS3
- Boostrap v5.2
- React.js
- Material UI
- Joi
- Axios
- Postman
- Fontawesome



## Installation 

Use npm package manager 

```bash
npm install --force
```

## Start Project
```bash
npm start
```
